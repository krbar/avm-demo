# AVM/Bicep Hands-on Session

This repository contains lab files and exercises for an Azure Verified Modules (AVM) and Bicep hands-on training session. The materials guide participants through Infrastructure as Code (IaC) best practices using Bicep templates and Azure Verified Modules.
